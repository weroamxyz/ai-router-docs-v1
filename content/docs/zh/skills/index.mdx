## 概述

**Skills** 是一种轻量级扩展协议，让 AI 编辑器（如 Claude Code、Codex、OpenClaw（龙虾）、Cursor、Windsurf 等）能够在编码环境内直接与 算力仓 交互，无需切换到浏览器或管理面板。

算力仓 Skills 现已支持在 Claude Code、Codex、OpenClaw（龙虾）等工具中直接执行查询模型、管理令牌、查看余额，以及提问 算力仓 使用问题等操作。

算力仓 目前提供以下 Skills：

## 支持的 AI 编辑器

所有 Skills 均可在以下 AI 编码工具中使用：

| AI 编辑器 / 编码助手 | 类型 | 备注 |
|----------------------|------|------|
| **Claude Code** | 终端 AI 编程助手 | Anthropic 官方 CLI |
| **OpenClaw** | 自托管 AI 助手平台 | 支持 Skills，可在对话中直接调用 `/newapi` 指令 |
| **Cursor** | AI 原生代码编辑器 | 基于 VS Code |
| **Windsurf** | AI 代码编辑器 | Codeium 出品 |
| **Cline** | VS Code AI 扩展 | 开源 AI 编码代理 |
| **Codex CLI** | 终端 AI 编程助手 | OpenAI 官方 CLI |

> 任何支持 Skills 协议的 AI 工具都可以使用 算力仓 Skills。
