<Callout type="info" title="チャット設定オプション">
  Unode
  コンソールのシステム設定->チャット設定で、以下のショートカットオプションを追加できます。これにより、トークン管理ページからCherry Studioにワンクリックで入力するのに便利です。
</Callout>

    ```json
    { "Cherry Studio": "cherrystudio://providers/api-keys?v=1&data={cherryConfig}" }
    ```

<Callout type="info">
  🍒 Cherry Studio は、プロフェッショナルユーザー向けに設計された強力なデスクトップAIクライアントです。30以上の業界インテリジェントアシスタントを統合し、さまざまな作業シナリオのニーズを満たし、作業効率を大幅に向上させます。
</Callout>

## NewAPI 接続方法

### パラメータ入力

プロバイダータイプ：NewAPIがサポートするタイプ  
APIキー：NewAPIで取得  
APIアドレス：NewAPIサイトアドレス

ご自身でデプロイしたUnodeを使用するか、サービスプロバイダーが合法的なアップストリームライセンスとコンプライアンス義務を持つUnodeサービスであることを確認してください。出所不明のAPIアドレスやキーを本番環境に接続しないでください。
