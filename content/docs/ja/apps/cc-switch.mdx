## 主な機能

### 🔌 Provider 管理
- **ワンクリック切り替え** — Claude Code・Codex・Gemini の API 設定をワンクリックで切り替え、環境変数や設定ファイルの手動編集が不要
- **マルチエンドポイント対応** — Provider ごとに複数のエンドポイントを設定可能、API キー管理とレイテンシ速度テスト付き
- **4 層モデル設定** — Haiku / Sonnet / Opus / Custom の 4 段階で細かくモデルを設定

### 🛠️ MCP サーバー管理
- **アプリ横断の統合管理** — Claude / Codex / Gemini 3 つの MCP サーバーを単一パネルで管理
- **3 つのトランスポートタイプ** — stdio、HTTP、SSE（Server-Sent Events）に対応
- **自動同期** — 統合インポート・エクスポート + 双方向同期

### 💬 プロンプト管理
- **マルチプリセット** — 無制限のプリセットとクイック切り替え
- **クロスアプリ対応** — Claude（`CLAUDE.md`）、Codex（`AGENTS.md`）、Gemini（`GEMINI.md`）
- **Markdown エディタ** — CodeMirror 6 + リアルタイムプレビュー

### 🌐 マルチプラットフォーム対応
- **デスクトップアプリ** — Windows・macOS・Linux のネイティブインストーラー
- **Web バージョン** — ヘッドレスサーバー / SSH 環境向けのブラウザアクセス方式
- **CLI バージョン** — インタラクティブメニュー方式とコマンドライン方式の両対応

## NewAPI 接続方法

CC Switch は `ccswitch://` Deep Link プロトコルに対応しており、Unode のトークン管理ページからワンクリックで Provider 設定をインポートできます。

### 設定手順

1. **Unode のトークン管理ページで、対象トークンのドロップダウンメニューをクリック**
   メニューから **CC Switch** オプションを選択すると、CC Switch アプリが自動的に起動し、設定ダイアログが表示されます。

2. **ダイアログで設定を完了**
   ![CC Switch 設定ダイアログ](/assets/cc_switch/fill_dialog.png)

   各フィールドの説明：
   - **アプリ**：上部でアプリタイプを切り替え — **Claude** / **Codex** / **Gemini**、必要に応じて対象アプリを選択
   - **名前**：設定の名前を入力（例：`My Claude`）、CC Switch で識別・切り替えしやすくするため
   - **メインモデル**（必須）— デフォルトで使用するメインモデル
   - **Haiku モデル** — 軽量・高速モデル
   - **Sonnet モデル** — バランス型モデル
   - **Opus モデル** — 最高性能モデル

   すべてのモデルはドロップダウン選択式で、未選択時は「请选择模型」（モデルを選択してください）と表示されます。

3. **設定完了**
   **「打开 CC Switch」**（CC Switch を開く）をクリックして設定を CC Switch にインポートし使用開始。**「取消」**（キャンセル）をクリックで操作を中止。

## インストール方法

### macOS（Homebrew 推奨）

```shell
brew tap farion1231/ccswitch
brew install --cask cc-switch
```

### Windows

[Releases](https://github.com/farion1231/cc-switch/releases) から `.msi` インストーラーまたはポータブル版 `.zip` をダウンロード。

### Linux

[Releases](https://github.com/farion1231/cc-switch/releases) から `.deb` パッケージまたは `.AppImage` をダウンロード。

ArchLinux ユーザー：

```shell
paru -S cc-switch-bin
```

### Web バージョン（ヘッドレス / SSH サーバー）

```shell
wget https://github.com/farion1231/cc-switch/releases/latest/download/cc-switch-web-linux-x64.tar.gz
tar -xzf cc-switch-web-linux-x64.tar.gz
cd cc-switch-web/
./cc-switch-web
```

デフォルトポート `17666`、ブラウザで `http://localhost:17666` にアクセス。
