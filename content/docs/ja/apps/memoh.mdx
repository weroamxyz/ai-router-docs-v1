<Callout type="info">
  Memoh
  はオープンソースのセルフホスト型AIエージェントプラットフォームです。各ボットは独立したコンテナ内で動作し、永続メモリと専用ファイルシステムを備えています。Telegram、Discord、飛書（Lark）、QQ、Matrix、企業微信、WeChat、メール、内蔵
  Web UI を含む9つのチャネルに対応し、MCPツール呼び出し、ブラウザ自動化、スケジュールタスクなど豊富なエージェント機能をサポートしています。
</Callout>

- 公式サイト：[https://memoh.sh](https://memoh.sh)
- 公式ドキュメント：[https://docs.memoh.ai](https://docs.memoh.ai)
- GitHub：[https://github.com/memohai/Memoh](https://github.com/memohai/Memoh)

## 主な機能

- **コンテナ分離**：各ボットは独立した containerd コンテナ内で動作し、専用のファイルシステムとネットワークを持ち、スナップショットやデータのインポート/エクスポートに対応
- **メモリエンジン**：LLM ベースの事実抽出、ハイブリッド検索（密 + 疎 + BM25）、24時間コンテキスト読み込み、メモリの圧縮と再構築
- **マルチチャネル対応**：Telegram、Discord、飛書（Lark）、QQ、Matrix、企業微信、WeChat、メール、Web UI
- **MCP 対応**：MCP プロトコル完全サポート（HTTP / SSE / Stdio / OAuth）、ボットごとに独立した MCP 接続管理
- **ブラウザ自動化**：Playwright 駆動のヘッドレスブラウザを内蔵し、Web 閲覧、フォーム入力、スクリーンショットなどの操作に対応
- **Web 管理パネル**：Vue 3 + Tailwind CSS ベースのモダンな管理画面。ストリーミングチャット、ツール呼び出しの可視化、ファイル管理などに対応

## クイックインストール

Memoh は Docker でデプロイします。ワンクリックインストール（Docker が必要）：

```bash
curl -fsSL https://memoh.sh | sudo sh
```

手動インストール：

```bash
git clone --depth 1 https://github.com/memohai/Memoh.git
cd Memoh
cp conf/app.docker.toml config.toml
# config.toml 設定ファイルを編集
sudo docker compose up -d
```

起動後、`http://localhost:8082` にアクセスしてください。デフォルトの認証情報：`admin` / `admin123`。

## NewAPI 接続方法

Memoh は OpenAI 互換のあらゆるモデルプロバイダーに対応しており、NewAPI を通じてさまざまな AI モデルサービスを一元管理・利用できます。

### 設定手順

#### NewAPI API Key の取得

NewAPI に登録・ログインした後、上部ナビゲーションバーの「コンソール」をクリックし、「トークン管理」から「トークン追加」ボタンをクリックして新しい API Key を作成します。適切な権限を選択し、「作成」をクリックしてください。

作成完了後、コピーボタンをクリックして、生成された API Key をコピーします。

![copy-api-key](/assets/memoh/copy_api_key.png)

#### Memoh でモデルプロバイダーを追加

Memoh Web 管理パネルにログインし、プロバイダー管理ページに移動して、NewAPI をクリックします。

![switch-to-newapi-provider](/assets/memoh/switch_to_newapi_provider.png)

設定ページで以下の情報を入力します：

- **API Base URL**：NewAPI のエンドポイント URL を入力します。例：`https://api.example.com/v1`（ローカルデプロイの場合は `http://localhost:3000/v1`）
- **API Key**：NewAPI からコピーした API Key を貼り付けます

「保存」をクリックして、プロバイダーの設定を完了します。

#### モデルのインポート

プロバイダーの設定完了後、モデル管理ページで自動インポートまたは手動で使用したいモデルを追加します。

#### ボットにモデルを設定

ボット設定ページに移動し、モデル設定セクションでデフォルトのチャットモデルを NewAPI プロバイダー経由で追加したモデルに切り替え、「保存」をクリックします。

![set_chat_model](/assets/memoh/set_chat_model.png)

以上で、NewAPI を Memoh のモデルプロバイダーとして設定が完了しました。Memoh の各チャネル（Telegram、Discord、飛書など）を通じて AI ボットと会話でき、すべてのリクエストは NewAPI 経由でルーティングされます。
