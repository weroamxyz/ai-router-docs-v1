## 概要

**Skills** は軽量な拡張プロトコルで、AI エディター（Claude Code、Codex、OpenClaw、Cursor、Windsurf など）がコーディング環境内で Unode と直接やり取りできます。ブラウザや管理パネルに切り替える必要はありません。

Unode Skills は Claude Code、Codex、OpenClaw などのツールで、モデル照会、トークン管理、残高確認に加えて、Unode の使い方も直接質問できます。

Unode は現在以下の Skills を提供しています：

## 対応 AI エディター

すべての Skills は以下の AI コーディングツールで利用できます：

| AI エディター / コーディングアシスタント | 種類 | 備考 |
|----------------------------------------|------|------|
| **Claude Code** | ターミナル AI プログラミングアシスタント | Anthropic 公式 CLI |
| **OpenClaw** | セルフホスト型 AI アシスタントプラットフォーム | Skills に対応し、対話内で `/newapi` コマンドを直接実行可能 |
| **Cursor** | AI ネイティブコードエディター | VS Code ベース |
| **Windsurf** | AI コードエディター | Codeium 製 |
| **Cline** | VS Code AI 拡張機能 | オープンソース AI コーディングエージェント |
| **Codex CLI** | ターミナル AI プログラミングアシスタント | OpenAI 公式 CLI |

> Skills プロトコルをサポートするすべてのツールで Unode Skills を使用できます。
