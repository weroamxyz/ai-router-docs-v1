## Overview

ComputeVault is an AI API gateway and usage management system designed for legally authorized scenarios, providing unified infrastructure for your AI applications. It is primarily intended for personal use, internal team use, and enterprise private deployment.

<Callout type="info">
  ComputeVault seamlessly integrates with leading global AI service providers, including OpenAI, Anthropic, Google
  Gemini, DeepSeek, Midjourney, Suno, and 30+ other model services.
</Callout>

### Core Features

- **Unified Interface**: Within the scope of legally authorized upstream accounts and services, it provides a unified API entry point, compatible with OpenAI's standard format.
- **Intelligent Routing**: Multi-channel load balancing, automatic failover, and weighted random distribution.
- **Cost Management**: Internal cost allocation and enterprise client billing management capabilities.
- **Security Control**: Token permission management, model access control, and API call auditing.
- **Data Insights**: Real-time dashboards, usage statistics, and cost analysis.
- **Multi-tenant Architecture**: Adaptable for personal use, internal team use, and enterprise private deployment.
