Your feedback is highly valued and crucial for project improvement. If you encounter any issues during use, please report them to us via GitHub Issues.

## 📝 How to Submit an Issue

Please visit our GitHub Issues page: [ComputeVault Issues](https://github.com/Calcium-Ion/new-api/issues)

<Callout type="info" title="Checks Before Submitting an Issue">
  Before submitting a new issue, we recommend that you:
</Callout>

    1. Search to see if a similar issue already exists
    2. Ensure you are using the latest version
    3. Check if there are already relevant answers in the [FAQ document](/en/docs/support/faq)

![feedback-issues](/assets/issues.png)
