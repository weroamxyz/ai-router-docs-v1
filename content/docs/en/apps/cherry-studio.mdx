<Callout type="info" title="Chat Settings Options">
  In the ComputeVault
  console's System Settings -> Chat Settings, you can add the following quick options for one-click filling into
  Cherry Studio on the token management page:
</Callout>

    ```json
    { "Cherry Studio": "cherrystudio://providers/api-keys?v=1&data={cherryConfig}" }
    ```

## NewAPI Integration Method

### Parameter Configuration

Provider Type: Types supported by NewAPI
API Key: Obtained from NewAPI
API Address: NewAPI Site Address

Please use your self-deployed ComputeVault, or ensure that the service provider has a ComputeVault service with legal upstream authorization and compliance obligations. Do not integrate API addresses or keys from unknown sources into a production environment.
