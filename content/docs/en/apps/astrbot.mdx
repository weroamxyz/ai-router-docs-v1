<Callout type="info">
  AstrBot is an open-source, all-in-one Agent chatbot platform that seamlessly integrates large language model capabilities into mainstream instant messaging software such as QQ, Feishu, DingTalk, and WeChat Work, creating a reliable and scalable conversational AI infrastructure for individuals, developers, and teams. Whether it's a personal AI companion, intelligent customer service, automation assistant, or enterprise knowledge base, AstrBot can quickly build production-ready AI applications within your instant messaging platform's workflow.

## NewAPI Integration Method

AstrBot supports integrating NewAPI as a model provider, allowing users to access and utilize various AI model services through NewAPI.

### Configuration Steps

#### Obtain NewAPI API Key

After registering and logging into NewAPI, click 'Console' in the top navigation bar, then click 'Token Management', and then click the 'Add Token' button to create a new API Key. Select the appropriate permissions and click 'Create'.

![create-api-key](/assets/astrbot/image.png)

After successful creation, click the 'Copy Key' button to copy the generated API Key.

![copy-api-key](/assets/astrbot/image-1.png)

#### Configure NewAPI Service Provider in AstrBot

Open the AstrBot administration panel, go to the 'Model Providers' page, then click the 'Add Model Provider' button.

NewAPI perfectly supports OpenAI Chat Completion and Responses interfaces. Click 'OpenAI' to enter the OpenAI provider configuration page.

In the pop-up dialog, set the API Base URL to the NewAPI interface address. Please use your own deployed Unode, or ensure that the service provider has legitimate upstream authorization and compliance obligations for the Unode service. If you have deployed NewAPI locally, fill in the local address, for example, `http://localhost:3000/v1`. Do not connect API addresses or keys from unknown sources to a production environment.

Then, enter the API Key into the 'API Key' field and click the 'Save' button.

![astrbot-provider-config](/assets/astrbot/image-2.png)

Then click 'Save' to complete the NewAPI provider configuration.

#### Apply Service Provider

Go to the 'Configuration File' page, find the 'Model' section, change the 'Default Chat Model' to the NewAPI provider you just created, and click the 'Save' button.

![apply](/assets/astrbot/image-3.png)

At this point, you have successfully configured NewAPI as AstrBot's model provider. Now, you can access and use various AI model services provided by NewAPI through AstrBot.
